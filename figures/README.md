# Miscellaneous figures

This directory contains mutliple notebooks that generates informative figures used in the paper.

- [`analyse_encoder_perf_on_dbaasp`](analyse_encoder_perf_on_dbaasp.ipynb): Analyzes the performance of our encoder on the DBAASP dataset, an AMP dataset different from the training one.
- [`dataset_complexities`](dataset_complexities.ipynb): Evaluate the complexity distribution (Shannon entropy) of different AMP datasets and subsets.
- [`ideal_identity.ipynb`](ideal_identity.ipynb): Empirically find the ideal splitting threshold
- [`latent_space_visualization`](latent_space_visualization.ipynb): Visualize the latent space of our encoder using UMAP.
- [`Visualize_data_annotations`](Visualize_data_annotations.ipynb): Visualize the correlation between annotations for different strains within the same specie
